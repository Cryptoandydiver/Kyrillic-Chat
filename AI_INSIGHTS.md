- OpenAI доля рынка: **86% → 46%** (потеря за 2 года)
- Claude обогнал OpenAI в B2B: **34% vs 32%** (данные Ramp)

- Gallup: молодёжь использует AI чаще, но понимает его хуже

## ДЕТАЛЬНЫЕ ВОРКФЛОУ

### 1. AI Knowledge Base (Obsidian + LLM)
**Автор:** участник чата (Oleg)
**Репозиторий:** `github.com/olegvg/kb-for-agents`
**Как работает:**
- Трёхуровневая структура: Sources → Summaries (.kb/summaries/) → Indexes (.kb/indexes/)
- Entry point: `.kb/indexes/_root.md`
- Запрос через `/kb-query` → AI читает indexes → descends в summaries → ответ с цитатами
- Без галлюцинаций, без RAG-инфраструктуры, без векторных БД
**Команды:** kb-build, kb-query, kb-refresh, kb-check-drift, kb-record-decision
**Формат хранения:** просто .md файлы внутри репозитория

### 2. Voice-to-Code Pipeline
- TG бот транскрибирует голосовые сообщения
- Промпты отправляются в Claude Code
- Статистика: 76 голос/нед, 119 мин всего, ~1.5 мин средняя

### 3. Compliance Review через LLM
- Проверка сотен рекламных креативов/мес для финтеха
- 95%+ проходимость
- Compliance officer: недели → дни

### 4. AI для медицинской диагностики
- Claude Opus 4.7 проанализировал анализы + симптомы + назначения
- Нашёл ошибку в антибиотике (неправильный класс и длительность)
- Врач подтвердил

### 5. AI для переговоров
- Контракт загружен в ChatGPT
- Результат: $80k → $30k/год

### 6. AI для визовых документов
- Claude Code проверил 62 документа
- Нашёл обременение на недвижимости
- Предотвратил отказ в визе

## AI-СКЕПТИЦИЗМ (важно)

Из постов Кирилла и обсуждений:

1. **«AI как Arduino / 3D printers»** — легко сделать прототип, сложно найти repeatable scalable business
2. **«Token consumption = новые eyeballs»** — метрика vanity, не revenue
3. **«93% AI-текста детектится»** — UMD/DeepMind: детекция по структуре сюжета
4. **«Vibecoding: +300–500% кода, +20% релизов»** — bottleneck сместился с написания на понимание/деплой
5. **«Shadow IT от no-code»** — бизнес-юниты сами ставят AI, создавая техдолг
6. **«AI не увеличивает прибыль компаний»** — Ramp: spending вырос 13×, P&L impact unclear
7. **«AI-стартапы = плохой бизнес»** — экономика токенов играет против них

## ВЫВОДЫ ДЛЯ ВАШЕГО БИЗНЕСА

**Из AI-скептицизма можно сделать сильное позиционирование:**
- «Я не продаю AI ради AI. Я продаю результат. Если AI не нужен — я скажу»
- «Vibecoder сделает демку за выходные. Я сделаю систему, которая не упадёт через месяц»
- «Большинство AI-проектов не окупаются. Я могу показать математику до, а не после»

**Инструменты, которые стоит внедрить самому:**
- kb-for-agents для своей базы знаний
- Voice-to-code для ускорения работы
- Compliance review как услуга для финтеха (если есть доступ к рынку)

**Что продавать:**
- Не «агента», а «аналитика 24/7 с доменной экспертизой»
- Не «внедрение AI», а «ROI от автоматизации»
- Не «подписку на инструмент», а «экономию $Xk/мес»
