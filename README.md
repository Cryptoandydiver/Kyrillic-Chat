# Kyrillic Chat

Анализ экспорта чата @kyrillic (ноябрь 2021 — июль 2026) + бизнес-инфраструктура solo AI-продуктов.

## Сайт (GitHub Pages)

**→ https://cryptoandydiver.github.io/Kyrillic-Chat/**

Продающий лендинг AI Knowledge Base as a Service.

## Файлы

| Файл | О чём |
|------|------|
| **CONTEXT.md** | Цель бизнеса, этапы, архитектура, ключевые решения |
| **SALES_PLAN.md** | План продаж простым языком: кому, что, сколько |
| **SUMMARY.md** | Полный обзор чата: идеи для заработка, AI-инструменты, инвестиции |
| **BUSINESS_IDEAS.md** | 15+ конкретных идей для бизнеса с ценами и ЦА |
| **AI_INSIGHTS.md** | AI-инструменты из чата, статистика рынка |
| **skills/** | Шаблоны для продаж, установки, поддержки (IP бизнеса) |
| **agents/trading-agent/** | Первый AI-агент (портфолио) |
| **products/ai-knowledge-base/** | Описание продукта + скрипт установки |

## Как читать

- Хотите понять бизнес → **CONTEXT.md** + **SALES_PLAN.md**
- Ищете идеи для заработка → **BUSINESS_IDEAS.md**
- Интересуетесь AI-инструментами → **AI_INSIGHTS.md**
- Полный обзор чата → **SUMMARY.md**
