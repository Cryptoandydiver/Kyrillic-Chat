## 1. ИДЕИ ДЛЯ ЗАРАБОТКА (ТОП-15)

- Обрабатывает **30k+ товаров/день** для **40+ магазинов**
- Находит скидки, пишет адаптеры, категоризирует
- Заменил **3 сотрудников** (операционка)

- Комментатор заработал больше, отговаривая компании от AI, чем внедряя его
- Стратегия: пилот → ROI → сравнение with/without

### 1.3 Платное AI-сообщество
- Подписка + еженедельные вебинары + AI-лекции
- Команда 3 человека
- Аудитория: предприниматели и менеджеры

- **Огромный неосвоенный рынок**

## 2. AI ИНСТРУМЕНТЫ И ВОРКФЛОУ

### Рейтинг популярности в чате:
1. **Claude Code** — лучший для кода и агентов
2. **ChatGPT** — 46% рынка B2C (было 86%), всё ещё доминирует
3. **DeepSeek** — быстро растёт, «тупой но дешёвый»
4. **Gemini** — догоняет ChatGPT за счёт Google дистрибуции
5. **Perplexity** — для исследований
6. **Grok** — упоминается, но не в активном использовании
7. **Cursor / Windsurf / Replit** — для vibecoding

### Детальные воркфлоу:

**AI Knowledge Base (Obsidian + LLM)**
- `github.com/olegvg/kb-for-agents` — трёхуровневая БЗ (sources → summaries → indexes)
- Парсит PDF в markdown, строится в виде .kb/ папки внутри проекта
- `/kb-query` — ответы с цитатами из документов, без галлюцинаций
- Не требует инфраструктуры (векторные БД не нужны)

**Voice-to-Code Pipeline**
- TG бот → транскрипция голосовых → промпты в Claude Code
- Статистика: 76 голосовых/нед, 119 минут, ~1.5 мин средняя

**Compliance Review Automation**
- LLM проверяет сотни креативов/мес для финтеха
- 95%+ проходимость
- Compliance officer: недели → дни

**Медицинская диагностика через AI**
- Загрузил анализы + симптомы + назначения в Claude Opus 4.7
- AI нашёл неправильный антибиотик (не тот класс, не та длительность)
- Врач подтвердил

**Переговоры через AI**
- Загрузил контракт в ChatGPT
- Снизил цену с $80k до $30k/год

**Визовые документы через Claude Code**
- Проверил 62 документа
- Нашёл обременение на недвижимость, о котором заявитель не знал
- Предотвратил отказ в визе

### AI-скептицизм (важно для реалистичной оценки):
- «AI как Arduino / 3D принтеры — легко прототипировать, сложно найти коммерчески масштабируемую ценность»

### Правила инвестиций (из чата):
- T-bills: 5%+ без риска. Любая доходность >2% над ставкой ФРС = gambling
- Долгосрочные сбережения — только в USD
- «$10k savings = $50/month forever» — упрощает финансовые решения
- Следить за крупными категориями, не за каждой копейкой
- «Live in the red is OK» — тратить больше дохода месяцами нормально, если есть резервы
